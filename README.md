# Mini Proje – Film Analizi ve Puan Filtreleme

Bu proje, Python kullanarak bir film listesini puanlarına göre filtreleyen basit ama öğretici bir uygulamadır.

## Kullanılan Konular
- Dictionary yapıları
- Döngüler
- Koşullar
- List Comprehension
- Fonksiyon kullanımı
- Veri işleme mantığı

## Proje Amacı
Python öğrenme sürecimde hem pratik yapmak hem de GitHub profilimi doldurmaya başlamak için oluşturduğum bir mini projedir.

Bu çalışma ile:
- Veriyi liste/dict içinde tutmayı,
- Filtrelemeyi,
- Kullanıcıya temiz çıktılar vermeyi,
- Kodları modüler hale getirmeyi öğreniyorum.

## Çalıştırmak İçin
python film_projesi.py
